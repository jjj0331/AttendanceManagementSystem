### スキル

| スキル | コマンド | Phase | 説明 |
|--------|---------|-------|------|
| プロジェクト概要 | `/project-overview` | 0 | プロジェクト概要書生成 |
| ディスカバリーセッション | `/discovery-session` | 0.5 | ドメイン知識抽出（必須） |
| イベントストーミング | `/event-storming {コンテキスト名}` | 1 | ES図生成 |
| ドメインモデリング | `/domain-modeling {集約名}` | 2 | ドメインモデル生成 |
| API設計 | `/api-design {集約名}` | 3 | API設計生成（並列実行可） |
| DB設計 | `/db-design {集約名}` | 3 | DB設計生成（並列実行可） |
| 画面設計 | `/screen-design {画面名}` | 3 | 画面設計生成（並列実行可） |
| コンテキスト間連携設計 | `/cross-context-design` | 3.5 | 連携設計書生成 |
| 設計レビュー | `/design-review` | - | 実装可能性を15項目で評価 |
| 工数見積もり | `/estimate` | - | 設計メトリクスから工数見積もり |
| スキャフォールド | `/scaffold` | 4 | プロジェクト構成生成 |
