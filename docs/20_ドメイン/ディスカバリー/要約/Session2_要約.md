# セッション2 要約 - 例外処理・制約深掘り

## 概要

**目的**: ホットスポット解消、業務ルールの数値化、例外処理フロー
**達成度**: 90%

---

## 抽出された要素

### 勤務パターン（3種）

| パターン | 所定時間 | 残業計算 |
|---------|---------|---------|
| 固定時間制 | 9:00-18:00（休憩1h、所定8h） | 所定終業後 |
| シフト制 | 早番/遅番/夜勤（各休憩1h） | 所定終業後 |
| フレックスタイム制 | コア10:00-15:00 | 月単位清算 |

### 36協定アラート閾値

| 区分 | 閾値 | 通知先 |
|------|------|-------|
| 月次注意 | 30h | 上長 |
| 月次警告 | 40h | 上長 + 人事 |
| 月次上限 | 45h | 上長 + 人事 + 本人 |
| 年次注意 | 300h | 上長 + 人事 |
| 年次上限 | 360h | 上長 + 人事 + 本人 |
| 特別条項月次 | 100h未満 | - |
| 特別条項年次 | 720h以内 | - |
| 特別条項回数 | 年6回まで（月45h超） | - |
| 特別条項平均 | 2-6ヶ月平均80h以内 | - |

### 有給管理ルール

| ルール | 数値 |
|-------|------|
| 初回付与 | 入社6ヶ月後に10日 |
| 最大付与 | 年20日（勤続6.5年以上） |
| 時効 | 付与日から2年 |
| 消化順序 | FIFO（古いものから） |
| 取得義務 | 年5日（基準日から1年以内） |
| 半日有給 | 0.5日単位（午前/午後） |
| 時間単位有給 | 1h単位、年5日分（40h）上限 |

### 割増率

| 区分 | 割増率 |
|------|-------|
| 時間外 | ×1.25 |
| 深夜（22:00-5:00） | ×1.25 |
| 時間外+深夜 | ×1.50 |
| 休日 | ×1.35 |
| 休日+深夜 | ×1.60 |

## 例外処理

| 例外 | 発生率/条件 | 対応 |
|------|-----------|------|
| 打刻忘れ | 全打刻の約8% | 翌日以降に修正申請 |
| 承認遅延 | 月末に集中 | 締め3営業日前リマインド、1営業日前督促 |
| 有給付与ミス | 入社日起算で全員バラバラ | 自動付与で対応 |
| システム障害 | - | 紙打刻簿→復旧後一括登録 |
| 月次締め中断 | - | 仮締め凍結→復旧後冪等に再実行 |
