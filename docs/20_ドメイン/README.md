# ドメイン

ビジネスドメインの知識を格納します。

---

## 構成

| ドキュメント | 説明 | 生成スキル |
|-------------|------|-----------|
| プロジェクト概要.md | プロジェクトの企画書 | `/project-overview` |
| ディスカバリー/ | セッション記録・ES用サマリー | `/discovery-session` |
| イベントストーミング/ | ES図（Mermaid形式） | `/event-storming` |
| {コンテキスト名}/{集約名}集約.md | ドメインモデル | `/domain-modeling` |
| コンテキストマップ.md | コンテキスト間の関係図 | ES結果から導出 |
| 用語集.md | ユビキタス言語 | ディスカバリーから導出 |

> **注**: 上記ドキュメントは各スキル実行時に自動生成されます。手動でファイルを作成する必要はありません。

---

## コンテキスト別集約

`/domain-modeling` スキルで生成された集約ドキュメントは、コンテキスト別のディレクトリに格納されます。

```
20_ドメイン/
├── {コンテキスト名}/
│   ├── {集約名}集約.md
│   └── ...
└── ...
```

---

## 集約ドキュメントの作成方法

`/domain-modeling` スキルを使用して作成します。

```bash
/domain-modeling {集約名}
/domain-modeling {集約名} --context {コンテキスト名}
```
