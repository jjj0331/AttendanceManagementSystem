# 運用

システムの運用に関するドキュメントを格納します。

---

## 構成

```
70_運用/
├── デプロイ/              # デプロイ手順・設定
├── 監視/                  # 監視・アラート設定
├── インシデント対応/       # 障害対応手順
└── ランブック/            # 定型運用作業
```

---

## ドキュメント一覧

| ドキュメント | 説明 | 作成タイミング |
|-------------|------|---------------|
| デプロイ手順 | 環境別のデプロイ方法 | 開発環境構築時 |
| 環境設定 | 環境変数・設定ファイル | 開発環境構築時 |
| 監視設計 | メトリクス・アラート定義 | 本番リリース前 |
| インシデント対応 | 障害時の対応フロー | 本番リリース前 |
| ランブック | 定型作業の手順書 | 運用開始後 |

---
