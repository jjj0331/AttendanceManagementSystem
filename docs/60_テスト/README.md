# テスト

テスト計画・テストケースを格納します。

## 推奨ドキュメント

- `テスト方針.md` - テスト戦略・カバレッジ目標
- `E2Eテストシナリオ.md` - エンドツーエンドテストケース
- `テストデータ.md` - テスト用データ定義

## テストピラミッド

```
    /\
   /E2E\
  /------\
 /Integration\
/--------------\
/     Unit      \
------------------
```

- **Unit**: ドメインロジック（集約、値オブジェクト）
- **Integration**: リポジトリ、外部サービス連携
- **E2E**: ユーザーシナリオ
