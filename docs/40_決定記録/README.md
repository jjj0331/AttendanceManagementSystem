# 決定記録 (ADR)

Architecture Decision Records - アーキテクチャ・設計に関する重要な決定を記録する。

アーキテクチャ変更も設計変更も、重要な技術的決定はすべてここに記録します。

## 命名規則

- `ADR-XXX_タイトル.md` の形式
- XXXは3桁の連番（001, 002, ...）
