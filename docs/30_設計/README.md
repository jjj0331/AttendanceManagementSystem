# 設計

技術的な詳細設計ドキュメントを格納します。

> **Note:** システム全体のアーキテクチャは [アーキテクチャ/](../10_アーキテクチャ/) を参照してください。

---

## 構成

| ドキュメント | 説明 | 生成スキル |
|-------------|------|-----------|
| API/エンドポイント/{集約名}.md | API設計 | `/api-design` |
| データベース/{集約名}.md | DB設計 | `/db-design` |
| 画面/screens/{画面名}.md | 画面設計 | `/screen-design` |

> **注**: 上記ドキュメントは各スキル実行時に自動生成されます。
> 複数集約を並列実行する場合は `skills/shared/parallel-execution-guide.md` を参照。

---

## 生成コマンド

```bash
/api-design {集約名}             # API設計
/db-design {集約名}              # DB設計
/screen-map {集約名}             # 画面マップ
/screen-design {画面名}          # 画面設計
```

詳細: [AI駆動開発ワークフロー](../00_ガイド/AI駆動開発/ワークフロー.md)
