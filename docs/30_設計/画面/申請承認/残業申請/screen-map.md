| API | メソッド | 画面での用途 |
|-----|---------|-------------|
| `/overtime-requests/{id}` | GET | 詳細表示 |
| `/overtime-requests/{id}/actions/approve` | POST | 承認ボタン（上長） |
| `/overtime-requests/{id}/actions/reject` | POST | 却下ボタン（上長） |
| `/overtime-requests/{id}/actions/resubmit` | POST | 再申請ボタン（申請者） |
